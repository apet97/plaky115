# FieldValueChangeRequest

Represents request for item field value change.

## Example Usage

```typescript
import { FieldValueChangeRequest } from "plaky115-mcp/models";

let value: FieldValueChangeRequest = {
  value: {
    "value": "To do",
  },
};
```
