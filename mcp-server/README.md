# plaky115-mcp

Unofficial local MCP server for the Plaky public API, generated by Speakeasy and
extended with curated workflow tools.

## Run

```bash
export PLAKY115_API_KEY=...
npx --yes --package . -- mcp start
```

When no `--scope` is supplied, all allowed tools are mounted. Use `--mode
curated`, `--mode generated`, or `--mode all` to choose the tool surface. Use
`--scope read`, `--scope write`, or `--scope destructive` when a client should
gate operations by behavior.

## Curated Tools

- `plaky_search_docs`
- `plaky_workspace_context`
- `plaky_find`
- `plaky_list_items`
- `plaky_plan_mutation`
- `plaky_execute_workflow`

Curated responses are compact by default. Pass `includeRaw: true` when a client
needs the original Plaky API payload.
