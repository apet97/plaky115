# plaky115

Unofficial TypeScript SDK for the Plaky public API, generated by Speakeasy and
extended with a small hand-written DX module.

## Install

```bash
npm install plaky115
```

## Generated Client

```ts
import { Plaky115 } from "plaky115";

const client = new Plaky115({ apiKeyAuth: process.env.PLAKY115_API_KEY });
const spaces = await client.spaces.listSpaces({ page: 1, pageSize: 50 });

for await (const page of spaces) {
  console.log(page.result.data);
}
```

## DX Wrapper

```ts
import { createPlakyClient, fieldValues, statusField } from "plaky115/plaky";

const plaky = createPlakyClient({ apiKey: process.env.PLAKY115_API_KEY! });
const boards = await plaky.boards.listAll({ space: "Ops" });

await plaky.items.createSimple({
  space: "Ops",
  board: "Roadmap",
  title: "SDK smoke",
  fields: fieldValues({ Status: statusField("Done") }),
});
```

The wrapper adds async item/page iterators, `listAll` helpers, field schema
lookup, title/email/id resolvers, typed field-value helpers, transport option
merging, and normalized `PlakyApiError` classes. The raw generated client is
available as `plaky.raw`.
