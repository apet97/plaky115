# Alignment

Represents text alignment of the number within the cell.

## Example Usage

```typescript
import { Alignment } from "plaky115/models";

let value: Alignment = "LEFT";

// Open enum: unrecognized values are captured as Unrecognized<string>
```

## Values

```typescript
"LEFT" | "RIGHT" | "CENTER" | "NONE" | Unrecognized<string>
```
