# plaky115 CLI

Unofficial Go/Cobra CLI for the Plaky public API, generated by Speakeasy and
extended with friendlier workflow commands.

## Auth

```bash
export PLAKY115_API_KEY=...
```

The generated `PLAKY115_API_KEY_AUTH` variable and `--api-key-auth` flag remain
supported.

## Workflows

```bash
plaky115 doctor
plaky115 workspace map -o json
plaky115 spaces list
plaky115 boards list --space-id 123
plaky115 items list --space-id 123 --board-id 456
plaky115 items create-simple --space-id 123 --board-id 456 --title "New item" --dry-run
plaky115 comments add --space-id 123 --board-id 456 --item-id 789 --text "Note" --dry-run
```

Raw generated operation commands remain available under `spaces`, `boards`,
`items`, `item-comments`, `item-comment-reactions`, `teams`, and `users`.
